# AI Study Buddy

AI Study Buddy is a Streamlit-based AI learning web app built using Python, Google Gemini API, Google Colab, and ngrok.

## Features
- Explains any topic in simple language
- Gives real-life examples
- Generates 5-question quizzes
- Evaluates learner answers and gives feedback
- Uses reusable prompt templates and role prompting

## Tech Stack
Python, Streamlit, Google Gemini API, Google Colab, ngrok
